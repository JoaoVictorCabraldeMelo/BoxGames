
Sistema de compra de jogos via terminal, desenvolvido em Java, orientação a objetos, flexibilidade e manutenibilidade.
O sistema simula uma loja de jogos, permitindo listar jogos, adicionar ao carrinho, calcular total da compra, aplicar descontos e finalizar a compra — sem persistência de dados.


 🎯 Objetivos do Projeto

- Implementar um fluxo completo de compra de jogos via terminal. 
- Facilitar evolução futura
